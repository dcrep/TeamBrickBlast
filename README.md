# TeamBrick: BrickBlast Barrier Destruction
3-strike barrier destruction with satisfying feel

A simple brick-based barrier that requires 3 hits to destroy.

Requirement was for it to have a satisfying feel.

Controls: standard WASD + mouse move and left-click melee attack

'R' key reloads scene
